* Rock vs Mine Prediction
************************************
. Project Overview
    This project is a Machine Learning classification model that predicts whether an object is a Rock (R) or a Mine (M) using Sonar signal data.
    The model is trained using Logistic Regression on the Sonar dataset.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------
* Dataset Information
. Dataset: Sonar (Rock vs Mine)
. Total Samples: 208
. Features: 60 numerical attributes
. Target:
  R → Rock
  M → Mine
-------------------------------------->  
* Libraries Used
  NumPy
  Pandas
  Scikit-learn
-------------------------------------->
* Machine Learning Algorithm
  Logistic Regression

-------------------------------------->
* Steps Performed
 1. Loaded the dataset using Pandas
 2. Split the dataset into training and testing data
 3. Trained the model using Logistic Regression
 4. Evaluated model performance using Accuracy Score
------------------------------------------------------------------>
* Model Accuracy
  . The model achieves good accuracy on both training and testing data.
